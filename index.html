Welcome to iCal-Event-Creator-App

Project Information:

The project aims to allow for a GUI interface that can create .ics files with various specifications that most iCal calendars are unable to do including multi-day events and GPS location. It also contains various error checking mechanisms to prevent invalid .ics files and some XSS scripting prevention.
